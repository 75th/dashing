# Examples

## Dockerfile

```
$ mkdir dockerfile
$ cd dockderfile
$ cp ../dockerfile.json ./dashing.json
$ wget https://docs.docker.com/reference/builder/`
$ dashing build
```

## BusyBox

```
$ mkdir busybox
$ cd busybox
$ cp ../busybox.json ./dashing.json
$ wget http://www.busybox.net/downloads/BusyBox.html
$ dashing build
```
